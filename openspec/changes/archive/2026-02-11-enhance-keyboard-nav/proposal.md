# Change: 增强登录注册表单键盘导航体验

## 为什么
当前登录注册表单仅支持鼠标操作，用户在使用键盘操作时无法顺畅地在各输入框之间切换。添加 Tab 键导航可以提升键盘用户的操作效率，实现无鼠标登录注册体验。

## 什么变更
- 确保登录表单的 Tab 键切换顺序：账号输入框 → 密码输入框 → 登录按钮
- 确保注册表单的 Tab 键切换顺序（如果存在）：账号输入框 → 邮箱输入框 → 密码输入框 → 确认密码输入框 → 注册按钮
- 回车键仅在表单校验通过时触发登录/注册操作

## 影响
- 受影响的规范：auth
- 受影响的代码文件：
  - `apps/web/app/ui/login-form.tsx`
  - `apps/web/app/ui/register-form.tsx`（如果存在）
- 测试验证：手动测试 Tab 键导航和回车键提交行为
