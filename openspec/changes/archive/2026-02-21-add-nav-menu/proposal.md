# 添加展开导航菜单

## 概述

在 farming 游戏页面（`/dashboard`）添加一个可展开的导航菜单，允许用户快速切换到其他功能页面。

## 背景

当前 farming 游戏页面没有导航菜单，用户需要：
1. 先返回 dashboard overview 页面
2. 然后通过左侧边栏导航到其他页面

这个提案将在 farming 页面添加一个便捷的展开菜单，让用户可以直接从游戏页面切换到其他功能。

## 目标

- 在 farming 页面右上角添加一个菜单按钮
- 点击按钮展开导航菜单，显示所有可用的功能页面
- 支持快速切换到：概览、项目、需求、任务、缺陷、奖励、习惯、每日、设置等页面
- 保持与现有 SideNav 风格一致的设计

## 用户故事

**作为** 用户
**我希望** 在 farming 页面有一个菜单可以切换到其他页面
**以便** 我可以不用返回 overview 就能快速访问其他功能

## 验收标准

- [ ] 页面右上角显示菜单按钮
- [ ] 点击按钮展开导航菜单
- [ ] 菜单显示所有 dashboard 子页面
- [ ] 点击菜单项跳转到对应页面
- [ ] 菜单支持收起/展开动画
- [ ] 与现有 UI 风格保持一致
- [ ] `pnpm build` 成功

## 设计考虑

### 位置
- 页面右上角固定位置

### 样式
- 与 SideNav 保持一致的设计语言
- 使用现有的图标组件

### 交互
- 点击菜单按钮展开/收起
- 点击菜单项自动跳转并收起菜单

## 技术实现

创建 `NavMenu` 组件，包含：
- 菜单按钮（固定在右上角）
- 下拉菜单（显示所有页面链接）
- 动画效果（展开/收起）
