## ADDED Requirements

### Requirement: 项目类型筛选器 UI
系统 SHALL 在项目列表页面提供项目类型筛选器，允许用户按项目类型筛选项目。

#### Scenario: 显示项目类型筛选器
- **GIVEN** 用户在项目列表页面
- **WHEN** 页面加载完成
- **THEN** 系统显示项目类型筛选器
- **AND** 筛选器包含选项：全部、冲刺项目、慢燃项目

#### Scenario: 按类型筛选项目
- **GIVEN** 用户在项目列表页面
- **WHEN** 用户点击"冲刺项目"筛选器
- **THEN** 系统只显示类型为 `sprint-project` 的项目
- **WHEN** 用户点击"慢燃项目"筛选器
- **THEN** 系统只显示类型为 `slow-burn` 的项目

#### Scenario: 显示全部项目
- **GIVEN** 用户在项目列表页面
- **WHEN** 用户点击"全部"类型筛选器
- **THEN** 系统显示所有项目（不应用类型筛选）

---

### Requirement: 组合筛选支持
系统 SHALL 支持同时应用状态筛选和类型筛选，两个筛选器独立工作。

#### Scenario: 状态和类型组合筛选
- **GIVEN** 用户已选择状态筛选为"有风险"
- **AND** 用户选择类型筛选为"冲刺项目"
- **THEN** 系统只显示状态为 `at_risk` 且类型为 `sprint-project` 的项目

#### Scenario: 仅状态筛选
- **GIVEN** 用户选择状态筛选为"正常"
- **AND** 用户选择类型筛选为"全部"
- **THEN** 系统只显示状态为 `normal` 的项目（所有类型）

#### Scenario: 仅类型筛选
- **GIVEN** 用户选择状态筛选为"全部"
- **AND** 用户选择类型筛选为"慢燃项目"
- **THEN** 系统只显示类型为 `slow-burn` 的项目（所有状态）

#### Scenario: 筛选器独立高亮
- **WHEN** 状态筛选器被激活
- **THEN** 仅状态筛选器按钮高亮显示
- **AND** 类型筛选器保持普通状态（除非也被激活）
- **WHEN** 类型筛选器被激活
- **THEN** 仅类型筛选器按钮高亮显示
- **AND** 状态筛选器保持普通状态（除非也被激活）
