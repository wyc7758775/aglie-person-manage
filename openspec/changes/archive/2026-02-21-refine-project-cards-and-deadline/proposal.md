# Change: 项目列表卡片容器滚动、卡片尺寸与截止时间展示优化

## Why

项目列表页当前存在：超出内容时整页滚动而非卡片容器内滚动；项目卡片尺寸偏大、一屏可见数量少；截止时间无过期/临近视觉区分，用户难以快速识别紧急与已过期项目。

## What Changes

- **卡片容器内滚动**：列表内容超出视口时，仅卡片所在内容区域在容器内垂直滚动，不撑长整页；头部（标题、筛选、添加按钮）保持固定。
- **项目卡片尺寸缩小**：缩小项目卡片整体尺寸（如内边距、字号、头像/图标尺寸、进度条高度等），使一屏可展示更多卡片，具体数值由实现与设计决定。
- **截止时间状态展示**：在卡片上展示的截止时间按状态区分样式：
  - **已过期**：红色文字，并在时间前显示过期相关表情（如 ⏰ 或约定表情），颜色可随过期程度适当加深。
  - **即将截止**：黄色/琥珀色警告样式；可根据「距截止日剩余时间」与「当前进度」使颜色逐渐加深（如越近且进度越低则越醒目）。
  - **未临近**：保持中性样式（如灰色或默认文字色）。
  具体色值、阈值（如「即将」定义为 N 天内）和表情由实现与设计约定。

## Impact

- **Affected specs**: project-management（delta 全部 ADDED）
- **Affected code**:
  - `app/dashboard/project/page.tsx` — 卡片容器可滚动布局、卡片尺寸样式、截止时间计算与样式/表情
  - `app/ui/dashboard/section-container.tsx` — 内容区可滚动（max-height + overflow-y-auto）或由页面包一层可滚动区域
