## MODIFIED Requirements

### Requirement: 项目创建与编辑弹窗
系统 SHALL 提供优化的项目创建/编辑弹窗，支持 ESC 关闭、二次确认、优化的表单布局和更好的视觉层次。

#### Scenario: 表单字段排序
- **WHEN** 用户查看项目弹窗表单
- **THEN** 字段按以下顺序显示：
  1. 背景图（高度 190px，独立卡片样式）
  2. 项目名称
  3. 项目类型
  4. 优先级
  5. 积分与项目状态（并列）
  6. 开始时间
  7. 结束时间
  8. 描述
- **AND** 不显示目标和标签字段
- **AND** 背景图不遮挡其他表单内容

#### Scenario: 表单美观性
- **WHEN** 用户查看简化后的项目表单
- **THEN** 背景图高度为 190px，作为视觉焦点
- **AND** 背景图使用圆角边框卡片样式
- **AND** 各字段间距适中（space-y-6），保持视觉平衡
- **AND** 进度条和时间戳作为辅助信息，不干扰主表单

#### Scenario: ESC 关闭二次确认文案
- **GIVEN** 项目弹窗处于打开状态
- **AND** 用户已修改表单字段（编辑态）
- **WHEN** 用户按下 ESC 键或点击遮罩层
- **THEN** 显示确认对话框询问「确定放弃添加的内容吗？」
- **AND** 用户选择「确定」后关闭弹窗并丢弃修改
- **AND** 用户选择「取消」后保持弹窗打开并保留修改

#### Scenario: 积分与项目状态并列布局
- **WHEN** 用户查看积分和项目状态区域
- **THEN** 积分设置和项目状态选择器在同一行显示
- **AND** 采用两列网格布局
- **AND** 在小屏幕设备上自动堆叠为两行

#### Scenario: 积分设置区域优化
- **WHEN** 用户查看积分设置区域
- **THEN** 「积分」label 右侧显示「自动计算」勾选框
- **AND** 勾选框文字为「自动计算」（而非「根据优先级自动计算」）
- **AND** 勾选框旁边显示 tooltip 图标
- **AND** 鼠标悬停 tooltip 时显示提示：「根据项目优先级自动计算积分值」
- **AND** 勾选框默认处于选中状态

#### Scenario: 创建/保存按钮位置
- **WHEN** 用户查看弹窗底部操作区域
- **THEN** 「创建项目」或「保存」按钮位于弹窗底部右侧
- **AND** 按钮采用右对齐布局
- **AND** 在小屏幕设备上保持右对齐

#### Scenario: 时间显示格式
- **WHEN** 用户查看项目时间戳信息（创建时间/更新时间）
- **THEN** 时间格式仅显示年月日（如：2026/2/11）
- **AND** 不显示时分秒信息
- **AND** 使用本地化日期格式

#### Scenario: 编辑模式底部布局
- **GIVEN** 用户正在编辑已有项目
- **WHEN** 用户查看弹窗底部
- **THEN** 不显示底部操作栏（创建按钮）
- **AND** 时间戳区域直接贴近底部，无多余空白
- **AND** 新建模式仍显示底部操作栏和创建按钮

### Requirement: 动态主题色
系统 SHALL 支持动态主题色配置，默认主题色与登录界面提示框颜色保持一致。

#### Scenario: 默认主题色
- **WHEN** 系统初始化主题配置
- **THEN** 默认主色为 #f97316（orange-500）
- **AND** 与登录界面提示框颜色保持一致

#### Scenario: 按钮主题色应用
- **GIVEN** 主题配置已设置
- **WHEN** 按钮组件渲染
- **THEN** 按钮使用主题配置中的主色
- **AND** 主要操作按钮（创建/保存）使用主色背景
