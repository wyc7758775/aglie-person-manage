# Change: 项目弹窗 PRD 变更更新

## Why

根据最新优化的 PRD 文档，需要对项目弹窗进行以下调整：统一按钮主题色、优化表单字段排序、改进积分与项目状态布局、添加 tooltip 提示、调整创建按钮位置以及更新关闭确认文案。

## What Changes

- **按钮默认主题色**：改为登录界面提示框颜色（#f97316 / orange-500）
- **表单字段排序**：背景图 → 项目名称 → 项目类型 → 优先级 → 积分/项目状态 → 开始时间 → 结束时间 → 描述
- **积分与项目状态并列**：积分和项目状态在同一行显示，采用两列布局
- **积分区域优化**：勾选框文字从「根据优先级自动计算」缩短为「自动计算」，添加 tooltip 提示「根据项目优先级自动计算积分值」
- **创建按钮位置**：明确位于弹窗底部右侧，右对齐
- **关闭确认信息**：统一确认对话框内容为「确定放弃添加的内容吗？」
- **时间显示格式**：仅显示年月日（YYYY/MM/DD），去掉时分秒
- **编辑模式底部布局**：编辑项目时移除底部操作栏空白，时间戳直接贴近底部
- **背景图位置调整**：移动到项目名称上方，限制高度 190px，采用独立卡片样式
- **背景图视觉优化**：圆角边框设计，不遮挡其他表单内容，作为视觉焦点
- **移除字段**：删除目标和标签字段
- **表单美观性优化**：减少字段后调整间距，保持视觉平衡和层次感

## Impact

- **Affected specs**: project-management, ui-components
- **Affected code**:
  - `app/dashboard/project/components/ProjectModal.tsx` - 弹窗表单布局和交互
  - `app/components/ui/` - 主题色配置更新
  - `app/lib/i18n/dictionary.*.ts` - 更新确认对话框和 tooltip 翻译键
