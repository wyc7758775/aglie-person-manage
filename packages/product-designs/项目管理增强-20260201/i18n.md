# 项目管理增强 i18n 国际化翻译

> 基于 [项目管理增强-prd.md](./项目管理增强-prd.md) 生成，供开发接入 `app/lib/i18n/` 字典使用。

## Key 规范

- 格式：`project.自定义key`，如 `project.status.normal`、`project.startTime`
- 与现有 `project` 模块合并，key 使用 camelCase
- 本次变更涉及：状态、时间字段、描述占位符

## 变更说明

| 类型 | Key | 说明 |
|------|-----|------|
| 新增 | `project.status.normal` | 状态：正常 |
| 新增 | `project.status.at_risk` | 状态：有风险 |
| 新增 | `project.status.out_of_control` | 状态：失控 |
| 替换 | `project.startDate` → `project.startTime` | 开始日期 → 开始时间 |
| 替换 | `project.endDate` → `project.deadline` | 结束日期 → 截止时间 |
| 移除 | `project.autoSave` | 不再展示，可保留 key 供历史兼容 |
| 可选 | `project.form.descriptionPlaceholder` | 可更新为 MD 编辑器占位符 |

## 兼容性说明

- **状态**：原 `project.status.active`、`project.status.planning` 等可保留，用于历史数据展示或迁移期间的临时映射
- **时间**：若后端字段名仍为 `startDate`、`endDate`，i18n key 可新增 `startTime`、`deadline` 作为展示用，或直接替换原 key 的翻译文案
- **autoSave**：从 UI 移除后，key 可保留空字符串或删除引用，避免报错
