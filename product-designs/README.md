# 产品设计稿（Product Designs）

本文件夹用于存放产品需求文档（PRD）、i18n 国际化翻译文档及产品设计相关文档。

## 目录结构

每个需求单独一个文件夹，命名格式：`{需求名}-{YYYYMMDD}`。

```
product-designs/
├── 项目管理增强-20260131/
│   ├── prd.md
│   └── i18n.md
├── 积分系统-20260201/
│   ├── prd.md
│   └── i18n.md
└── README.md
```

- **需求名**：使用当前语言（如中文「项目管理增强」、英文「project-management-enhancement」）
- **日期**：创建需求时的年月日，格式 `YYYYMMDD`

## 文件说明

- **prd.md**：产品需求文档，描述功能背景、目标、用户故事、功能需求、交互逻辑及验收标准，含墨刀原型链接
- **i18n.md**：界面文案的中英日三语翻译表，Key 规范为 `模块.自定义key`，如 `project.name`

## 使用方式

在 Cursor 中可调用「产品需求文档指导」技能，以资深产品经理身份协助完成 PRD 的撰写与迭代。PRD 涉及界面文案时，可一并生成 i18n 翻译文档。
