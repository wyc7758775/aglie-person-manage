# 产品设计稿（Product Designs）

本文件夹用于存放产品需求文档（PRD）、i18n 国际化翻译文档及产品设计相关文档。

## 目录说明

- **PRD 文档**：产品需求文档，描述功能背景、目标、用户故事、功能需求、交互逻辑及验收标准
  - 命名：`{功能或模块名}-prd.md`，如 `积分系统-prd.md`
  - 含墨刀原型链接及交互逻辑表，供前端实现
- **i18n 翻译文档**：界面文案的中英日三语翻译表，供开发接入 `app/lib/i18n/` 使用
  - 命名：`{功能或模块名}-i18n.md`，如 `积分系统-i18n.md`
  - Key 规范：`模块.自定义key`，如 `project.name`

## 使用方式

在 Cursor 中可调用「产品需求文档指导」技能，以资深产品经理身份协助完成 PRD 的撰写与迭代。PRD 涉及界面文案时，可一并生成 i18n 翻译文档。
