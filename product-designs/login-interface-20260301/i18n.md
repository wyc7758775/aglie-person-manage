# 登录界面 i18n 国际化翻译

> 基于 [登录界面 PRD](./prd.md) 生成，供开发接入 `app/lib/i18n/` 字典使用。

## Key 规范

- 格式：`模块.自定义key`，如 `auth.login.title`
- 模块名固定为 `auth`，key 使用 camelCase
- 支持嵌套语义：`auth.login.emailPlaceholder`、`auth.login.error.invalidEmail`

## 生成说明

- **中文**：以 PRD 中的原文为准
- **English**：符合英文表达习惯，简洁明了
- **日本語**：使用敬体（です/ます調），符合日本商务应用习惯

## 使用示例

```typescript
// 在组件中使用
import { useTranslation } from '@/app/lib/i18n';

function LoginPage() {
  const { t } = useTranslation();
  
  return (
    <h1>{t('auth.login.title')}</h1>
    <input placeholder={t('auth.login.email.placeholder')} />
    <button>{t('auth.login.action.submit')}</button>
  );
}
```
