# 导航架构调整 i18n 国际化翻译

> 基于 [导航架构调整-prd.md](./prd.md) 生成，供开发接入 `app/lib/i18n/` 字典使用。

## Key 规范

- 格式：`模块.自定义key`，如 `projectDetail.backToProjectList`、`dashboard.nav.overview`
- 新增 `projectDetail` 模块用于项目详情页专属文案
- 导航项复用或引用现有 `dashboard.nav.*`，Tab 标签复用 `dashboard.nav.requirement`、`dashboard.nav.task`、`dashboard.nav.defect`
- key 使用 camelCase

## 变更说明

| 类型 | Key | 说明 |
|------|-----|------|
| 新增 | `projectDetail.*` | 项目详情页顶部栏、Tab、下拉菜单等文案 |
| 复用 | `dashboard.nav.overview` | 概览 |
| 复用 | `dashboard.nav.project` | 项目 |
| 复用 | `dashboard.nav.rewards` | 奖励 |
| 复用 | `dashboard.nav.notifications` | 通知 |
| 复用 | `dashboard.nav.setting` | 设置 |
| 复用 | `dashboard.nav.requirement` | 需求（Tab 标签） |
| 复用 | `dashboard.nav.task` | 任务（Tab 标签） |
| 复用 | `dashboard.nav.defect` | 缺陷（Tab 标签） |
| 复用 | `project.add` | 添加项目 / 新建项目 |
| 复用 | `project.edit` | 编辑项目 |
| 复用 | `project.delete` | 删除项目 |
| 复用 | `project.empty` | 空状态文案 |

## 兼容性说明

- **dashboard.nav**：需求、任务、缺陷从一级导航移除后，其 key 仍保留，用于项目详情页 Tab 标签
- **project.***：项目列表页的 add、edit、delete、empty 等沿用现有 key，若有差异可新增 `project.list.*` 子键
- **projectDetail**：全新模块，与现有 `project` 模块并列，避免覆盖已有文案

## 接入说明

开发时在 `app/lib/i18n/dictionary.zh.ts`、`dictionary.en.ts`、`dictionary.ja.ts` 中新增 `projectDetail` 对象，并按上表填充翻译。Tab 标签可直接使用 `t('dashboard.nav.requirement')` 等。
